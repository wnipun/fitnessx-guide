## System User Management

Manage user accounts that have access to the system. You can create or edit user accounts as required.

![User Management](settings/users.png)

### Creating a User Account

To create a new user account, click on the "New user" button. Fill in the required details. Provide a temporary password and share it with the user to log in for the first time. Please ensure that the user changes their password upon first login for security purposes.

![Create User Form](settings/create-user.png)

### User Groups

User groups help in managing permissions for different users. You'll be provided with an admin account by default. You can add additional users and assign them admin or staff roles. 

- **Admin**: Has full access to all system features and settings.
- **Staff**: Has limited access to specific features.
